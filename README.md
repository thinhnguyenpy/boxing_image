# boxing_image
code và hướng dẫn để boxing images cho YOLOV8

file extract_frames.py là dùng để extract ảnh từ video

các lệnh để tải tool boxing: 

pip3 install labelImg
labelImg
labelImg [IMAGE_PATH] [PRE-DEFINED CLASS FILE]

IMAGE_PATH: link tới folder chứa ảnh
PRE-DEFINED CLASS FILE: link tới classes.txt (ví dụ: labels/classes.txt)
